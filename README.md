##About

A super simple module for getting the [cpuid](https://en.wikipedia.org/wiki/CPUID) of a computer.


###Usage
Simply install:

    npm install asm-cpuid

Call the module:

     var cpuid = require("asm-cpuid");
    console.log(cpuid());

 Example output:

     [ 13, 132675, 1979931527, 0, 469228722 ]
